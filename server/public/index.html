API Documentation

  • GET /categories - Retrieves all categories.
  • POST /categories - Creates a new category.

  • GET /services - Retrieves all companies.
  • GET /services/category/:category - Retrieves all companies belonging to the specified category.
  • GET /services/:id - Retrieves a specific company by ID.
  • GET /services/:serviceId/bookings/date/:date - Retrieves all bookings for a specific company on the specified date.
  • POST /services - Adds a new company to the list. Ensure that all required fields are provided.
  • PUT /services/:id - Updates an existing company. Check if the company with the specified ID exists before updating.

  • GET /bookings/user/:email - Retrieves all bookings associated with a specific user's email address.
  • POST /bookings - Creates a new booking. Ensure that all fields are provided.
  • DELETE /bookings/:id - Deletes a specific booking.
