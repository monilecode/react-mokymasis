API Documentation

  • GET /categories - Gauna visas kategorijas.
  • POST /categories - Sukuria naują kategoriją.

  • GET /services - Gauna visas įmones.
  • GET /services/category/:category - Gauna visas įmones, priklausančias nurodytai kategorijai.
  • GET /services/:id - Gauna konkrečią įmonę pagal ID.
  • GET /services/:serviceId/bookings/date/:date - Gauna visus užsakymus konkrečiai įmonei nurodytą dieną.
  • POST /services - Prideda naują įmonę į sąrašą. Užtikrinti, kad būtų pateikti visi būtini laukai.
  • PUT /services/:id - Atnaujina esamą įmonę. Patikrinti, ar įmonė su nurodytu ID egzistuoja, prieš atnaujinant.

  • GET /bookings/user/:email - Gauna visus užsakymus, susijusius su konkretaus vartotojo el. pašto adresu.
  • POST /bookings - Sukuria naują užsakymą. Užtikrinti, kad būtų pateikti visi laukai.
  • DELETE /bookings/:id - Ištrina konkretų užsakymą.
